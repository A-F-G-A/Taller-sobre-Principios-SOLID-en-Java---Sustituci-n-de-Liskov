# Taller Principio de Sustitución de Liskov (LSP) en Java


## Objetivo
Aprender el **Principio de Sustitución de Liskov (LSP)**, que indica que las clases derivadas deben poder sustituir a sus clases base sin alterar la funcionalidad del programa. Se busca crear jerarquías de clases coherentes y polimórficas.
